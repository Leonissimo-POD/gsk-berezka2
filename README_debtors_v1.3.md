# Патч: финансовая дисциплина / должники v1.3

## Что загрузить в GitHub

1. Файл `debtors.html` — в корень репозитория.
2. Папку `data` — в корень репозитория.
3. Из файла `css_patch_for_debtors.txt` скопировать блок CSS в конец вашего файла `css/style.css`.

## Что внутри

Обезличенный список боксов-должников по взносам за 2025 год на 01.05.2026:
57 боксов.

На сайт не добавлены:
- ФИО;
- телефоны;
- адреса;
- персональные комментарии;
- точные суммы, если они не указаны в исходном изображении.

## Навигация

Чтобы пункт появился в меню всех страниц, добавьте ссылку в блок `<nav id="nav">`:

<a href="debtors.html">Должники</a>

Если не добавлять ссылку в меню, страница всё равно будет доступна по адресу:
`/debtors.html`
