## Как обновить сайт
1. Распакуйте ZIP.
2. Откройте репозиторий GitHub.
3. Нажмите Add file → Upload files.
4. Перетащите все файлы из папки.
5. Подтвердите Commit changes.
6. Подождите 1–5 минут, GitHub Pages обновит сайт.

## Где редактировать
- Тексты страниц: HTML-файлы
- Дизайн: css/style.css
- Форма обращения: appeals.html
