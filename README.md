# ГСК «Березка-2» — сайт для GitHub Pages

## Как опубликовать

1. Создайте аккаунт на GitHub.
2. Создайте новый репозиторий:
   - для основного бесплатного адреса: `ВАШ_ЛОГИН.github.io`
   - или для проектного сайта: `gsk-berezka2`
3. Загрузите в репозиторий все файлы из этой папки.
4. Откройте Settings → Pages.
5. Source: Deploy from a branch.
6. Branch: main, folder: /root.
7. Подождите 5–10 минут.

## Что редактировать

- Тексты страниц: файлы `.html`
- Цвета и внешний вид: `css/style.css`
- Форма обращения: ссылка в `appeals.html`
- Документы: папка `docs`
